## Assignment 3: WordPress Child Theme

### Child Themes

A child theme is to override existing functionality without modifying changes to the theme. The child theme prevents the original file from errors that affects results. A child theme is significant if there are additional changes needed in the script and you are not wanting the theme structure to result null.

### Template Hierarchy

Template Hierarchy is a process wordpress uses to check template files in precedent order. This ensures that the most specific template files are run first. For example, child-style.css would run first, before style.css. This is because the child theme is checked before the parent's theme.
